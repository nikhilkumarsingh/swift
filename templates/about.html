About this project

“SWIFT ARS (AIRLINE RESERVATION SYSTEM)” is a DBMS project aimed at simulating an airline reservation system.

  • Stack:

    • SQLite (RDBMS)
    • Flask (Web Framework in Python)
    • HTML, CSS, Bootstrap (Front-end)
    • APscheduler (Advanced scheduler in python for simulating flights)

  • Features:

    • Sign Up and Login functionality.
    • Automatic flight simulation.
    • Flight filtering on basis of origin, destination, date and company.
    • Profile section for user to check booked upcoming and passed flights.
    • Booking cancellation.
